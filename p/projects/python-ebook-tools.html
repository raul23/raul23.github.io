Project: python-ebook-tools

Github repo @ python-ebook-tools

ebook-tools is a very interesting project that consists of shell scripts for organizing and managing ebook collections. I decided then to port those useful shell scripts into Python so it can be easier to extend them with a simpler language like Python.

Currently, I am almost done porting the most important of the shell scripts: organize-ebooks.sh

Eventually, I would like to port all the scripts and integrate them through a GUI application so it can be easier for anyone to configure and run the scripts. Maybe add these scripts as a plugin to Calibre could be another possiblity instead of building a GUI app from scratch.
