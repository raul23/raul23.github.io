Interactive scatter plots

Here you will find the interactive scatter plots generated for the dev-jobs-insights project. Each figures are accompanied by a small table that presents some stats (e.g. published dates) about the data used for generating the corresponding figure.

Scatter plot of salaries in Europe

The y-axis represents the number of job posts used for computing the average mid-range salary for the given country, e.g. Germany is associated with an average mid-range salary of $70302 based on 34 job posts.

Table 1 Stats for the scatter plot of salaries in Europe

Number of jobs posts 103
Published dates 2018-09-18 to 2018-09-28
